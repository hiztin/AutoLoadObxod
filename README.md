# AutoLoadObxod
Вам тоже зачастую лень при включении компьютера включать обход? Мне тоже
# Итак, как сделать так что бы это работало и у вас тоже?
Всё просто, для начало замените в 3 строчке текст в ковычках на путь к файлу с вашим обходом, после чего сохраните.
Что бы посмотреть полный путь к файлу нажмите на него праавой кнопкой мыши, после нажав свойства.
Потом нажмите Win + R, введите shell:startup и закиньте ваш файл
**Важно:**
если ты переместишь папку с обходом или переименуешь её, автозагрузка отвалится, так как путь в скрипте станет неверным.
**Репозиторий безполезный для людей у которых эта функция есть в сборке, или для тех кто может сделать это сам.**
